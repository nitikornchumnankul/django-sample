# django-sample
### Create a Project
```
django-admin startproject django-sample
```
### Runserver
```
python manage.py runserver
```

### Create Application
```commandline
python manage.py startapp myapp
```
### Starting the Admin Interface
```commandline
python manage.py migrate
```
and
```commandline
python manage.py createsuperuser
```
